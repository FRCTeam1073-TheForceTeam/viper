資料匯入/出說明
========================

## USB 隨身碟

可以使用一個 USB 隨身碟，轉移偵查資料至另一個觀賽系統。

1. 將筆記型電腦連接至觀賽系統的乙太網路 (或其中一個 USB-C 網路卡)
2. 將 USB 隨身碟插入筆記型電腦。
3. 在筆記型電腦的瀏覽器中，開啟觀賽頁面：**`$URL$`**
4. 在首頁點擊活動。
5. 滑動到左欄的最底部，並點擊「匯出所有活動資料 (Export all event data)」
6. 點擊「下載 (Download)」連結。
7. 儲存 `.json` 檔案至 USB 隨身碟中。
8. 將 USB 隨身碟從筆記型電腦移除。
9. 將 USB 隨身碟插入另一個觀賽系統的裝置。
10. 在該裝置的瀏覽器中開啟觀賽系統。
11. 點擊首頁中的「+ 新增活動 (+ Add an event)」
12. 在「匯入 (Import)」區段，點擊「選擇檔案 (Browse...)」
13. 選擇 USB 隨身碟中的 `.json` 檔案並開啟。
14. 你將會在活動頁面中看到最新匯入的資料。

## 網路連線

當兩個觀賽系統皆連接至網際網路時，可以透過能存取兩者的瀏覽器轉移活動資料。

1. 在你的瀏覽器中，前往你想要匯出的活動 (即為資料的來源)。
2. 點擊左欄最底部的「匯出所有活動資料 (Export all event data)」連結。
3. 在「轉移活動 (Transfer Event)」下方，輸入另一系統的主機名稱。對於在本機執行的觀賽系統，主機名可能為「localhost」或帶有通訊埠號，如「localhost:8080」。
4. 點擊「轉移 (Transfer)」按鈕。
5. 你將會在活動頁面中，看到最新匯入的資料。
