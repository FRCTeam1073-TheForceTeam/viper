Instructions de reconnaissance
========================

## Instructions de reconnaissance

Observez votre robot et enregistrez ses actions. Utilisez les triangles orientés vers le haut pour augmenter les compteurs. Si vous faites une erreur, utilisez le bouton d'annulation.

Le jeu se joue en deux segments :

 - Autonome — Le robot agit de manière indépendante sans contribution des joueurs humains.
 - Télécommandé — Le robot est contrôlé par télécommande.

L'autonome et le telop ont leurs propres sections pour enregistrer les données, alors assurez-vous de basculer entre eux quand l'autonome se termine. Le bouton en bas de l'écran autonome commencera à clignoter comme rappel.

Enregistrez à la fois les emplacements où le robot collecte les pièces de jeu et où il les place. Les robots ne peuvent tenir qu'une seule pièce de jeu à la fois, vous allez donc passer du temps à enregistrer la collecte et la mise en place.

Utilisez l'un des boutons de la section finale pour sauvegarder les données une fois que vous avez terminé. Le bouton suggéré sera agrandi — généralement ce sera le bouton pour passer au match suivant, mais dans certains cas, l'application vous recommandera de télécharger vos données vers le centre de reconnaissance. Si vous utilisez une connexion filaire, branchez votre apareil sur le hub avant de télécharger vos données.
