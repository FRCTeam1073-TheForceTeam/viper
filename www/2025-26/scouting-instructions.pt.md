Instruções de Escutismo
========================

## Instruções de Escutismo

Observe o seu robô e registe as suas ações. Utilize os triângulos apontados para cima para aumentar os contadores. Se cometer um erro, utilize o botão de desfazer.

O jogo é jogado em dois segmentos:

 - Autônomo — O robô atua de forma independente sem contribuição dos jogadores humanos.
 - Teleoperado — O robô é controlado por controlo remoto.

O autônomo e o telop têm as suas próprias secções para registar dados, por isso certifique-se de que muda entre eles quando o autônomo termina. O botão na parte inferior do ecrã autônomo começará a piscar como lembrança.

Registe tanto os locais onde o robô recolhe as peças de jogo como onde as coloca. Os robôs só podem segurar uma peça de jogo por vez, pelo que estará a alternar entre registar a recolha e a colocação.

Utilize um dos botões na secção final para guardar os dados quando terminar. O botão sugerido será ampliado — normalmente este será o botão para avançar para a próxima partida, mas em certos casos a aplicação recomendará que carregue os seus dados para o centro de escutismo. Se estiver a usar uma ligação com fios, ligue o seu dispositivo ao hub antes de enviar os seus dados.
