## 匯入活動時程

### FRC

1. 前往 [The Blue Alliance](https://www.thebluealliance.com/) 或 [First Inspires FRC](https://frc-events.firstinspires.org/) 或 [First Inspires FTC](https://ftc-events.firstinspires.org/)
2. 前往你想要匯入的賽事時程頁面。
3. 使用 `Ctrl-u` 查看該頁面的網頁原始碼。
4. 使用 `Ctrl-a` 選取全部的原始碼。
5. 使用 `Ctrl-c` 複製所有選取的原始碼。

### FTC

1. 前往 [The Orange Alliance](https://www.theorangealliance.com/)
2. 前往你想要匯入的賽事時程頁面。
3. 以右鍵點擊頁面上的任意位置，並選擇「檢查 (Inspect)」。
4. 以右鍵點擊 `<html`，並選擇「複製 outerHTML (Copy outerHTML)」。

## 匯入隊伍清單

複製並貼上任意隊伍清單。非隊號資料 (例如球隊名稱) 將被忽略。隊號清單將產生隨機練習時間表，讓你可以在賽事的官方比賽時間表公佈之前開始進行 pit 偵察和照片偵察。
