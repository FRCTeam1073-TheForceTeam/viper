Instrucciones de Importación de Cronograma de Evento
========================

## Importar un Cronograma de Evento

### Para FRC

1. Ve a [The Blue Alliance](https://www.thebluealliance.com/) o [First Inspires FRC](https://frc-events.firstinspires.org/) o [First Inspires FTC](https://ftc-events.firstinspires.org/)
2. Navega a una página que muestre el cronograma del evento que deseas importar.
3. Usa `Ctrl-u` para ver el código fuente de la página del evento.
4. Usa `Ctrl-a` para seleccionar todo el código fuente.
5. Usa `Ctrl-c` para copiar todo el código fuente seleccionado.

### Para FTC

1. Ve a [The Orange Alliance](https://www.theorangealliance.com/)
2. Navega a una página que muestre el cronograma del evento que deseas importar.
3. Haz clic derecho en cualquier lugar de la página y elige "Inspeccionar"
4. Haz clic derecho en `<html` y elige "Copiar HTML externo"

## Importar una Lista de Equipos

Copia y pega cualquier lista de equipos. Los datos que no sean números de equipo (como nombres de equipos) serán ignorados. La lista de números de equipo generará un cronograma de práctica aleatorio que te permitirá comenzar la exploración de foso y exploración de fotos antes de que se conozca el cronograma de partido oficial para el evento.
